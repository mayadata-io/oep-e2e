## DMaaS

Data Management as a Service, this is supported for two ways
- Storage snapshot for cStor Volume
- Restic for Jiva / Local PV

DMaaS data is backed in following supported storage buckets. 
- AWS S3
- GCP Storage Bucket
- Minio
